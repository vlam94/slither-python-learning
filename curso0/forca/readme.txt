  como cola para ajudar na adaptação do codigo funcional

Regras do jogo:
    - O jogo se inicia com uma palavra secreta 
      contendo espaços (_) para cada letra    
    - Chute apenas uma letra por vez; não use  
      letras inválidas no português (ẽ,ò,î...) 
    - Você tem 8 chutes para acertar a palavra 
      e poderá adicionar 2 palavras ao jogo caso 
      vença a forca!
    - o modo hard corta pela metade o numero de
      chutes possíveis, mas permite adicionar
      uma palavra a mais em caso de vitória
